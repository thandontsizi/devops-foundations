# Networking Commands:
1. ip addr:
- Shows all network interfaces and their assigned IP addresses.
- Helps the device's IP on the network.

2. ip route:
- Displays the routing table, including default gateway.
- Shows how packets are sent to other networks.

3. ping:
- Sends ICMP packets to another machine to test connectivity and measure response time.

4. ss:
- Lists listening sockets and ports on the machine.
- Helps see which services are active.

5. curl:
- Connects to a server and retrieves data over protocols like HTTP/HTTPS.
- Tests communication at the application level.
