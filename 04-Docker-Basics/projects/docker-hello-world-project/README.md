# Docker Hello World Project:
## Objectives:
- Install Docker (verify installation: docker --version).
- Build simple Docker image.
- Run a container.
- Understand container lifecycle commands.

## Overview:
This project introduces basic Docker concepts: images, containers, Dockerfiles, and commands to manage them.

## Definitions:
- **Image**: A blueprint for creating containers.
- **Container**: A running instance of an image.
- **Dockerfile**: Text file with instructions to build an image.
- **Registry**: A repository to store Docker images (e.g., Docker Hub).

## Expected Outcomes:
- Verify Docker installation (docker --version).
- Build and run a simple container.
- Document commands and results.
- Complete self-test in tests.md.
