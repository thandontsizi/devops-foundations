# Docker Commands:
This file gives commands used  with brief explanations

## 1. Verify Docker Version and/or Installation:
- Version: docker --version 
- Installation Test: docker run hello-world

## 2. Build an Image: Build image from Dockerfile in current directory.
- docker build -t my-first-image .

## 3. List Images: List all Docker images.
- docker images

## 4. Run a Container: Run container interactively.
- docker run -it my-first-image
