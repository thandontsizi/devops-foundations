# Git Basics:
This folder contains the fundamentals of Git and GitHub for professional DevOps workflows.

## Goal:
- Understand Git as a version control system.
- Learn how to commit, branch, and push safely.
- Prepare for practical Git usage in collaborative environments.

## File Structure:
- README.md -> Concepts and explanations.
- commands.md -> Command cheat sheet.
- tests.md -> Questions to test understanding.
